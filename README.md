# **__Tableau Dashboards__**

A curated collection of interactive Tableau dashboards built for analytics, storytelling, and data exploration.

This repository contains multiple Tableau dashboards created for learning, portfolio development, and real‑world analytics use cases. Each dashboard includes:

- A clear business problem  
- Dataset source  
- Dashboard screenshots (optional)  
- Key insights  
- Instructions for reproduction  


## **__Dashboards Included__**

### **OTT Viewer Drop‑off & Retention Dashboard**  
Detailed Dashboard README: 
https://github.com/karthic180/tableau_dashboards/blob/main/README_OTT_Tableu.md
Dashboard PDF:  
https://github.com/karthic180/tableau_dashboards/blob/main/OTT_Dashboard_Tableau.pdf  

 **Dataset:**  
https://www.kaggle.com/datasets/eklavya16/ott-viewer-drop-off-and-retention-risk-dataset  

 **Purpose:**  
Analyze how viewers engage with OTT content at the episode level, identify drop‑off patterns, and understand retention risk.


## **__Features__**

- KPI tiles (Completion %, Drop‑off Probability, Retention Risk, Episode Duration)  
- Completion by Title  
- Drop‑off Probability by Episode  
- Retention Risk vs Completion (scatter)  
- Content Attribute Heatmap  
- Global filters (platform, genre, release year, season, title)  
- Click‑to‑filter interactivity
