##Tableau Dashboards Collection
A curated collection of interactive Tableau dashboards built for analytics, storytelling, and data exploration.

This repository contains multiple Tableau dashboards created for learning, portfolio development, and real‑world analytics use cases. Each dashboard includes:

A clear business problem

Dataset source

Dashboard screenshots (optional)

Key insights

Instructions for reproduction
